Setting up a Python Virtual Environment

Virtual environments are useful for setting up your Python libraries without impacting the whole system's Python installation (or when you don't have root access). To build an environment for Python 3 and install some libraries, run the following commands from the home directory:

python3 -m virtualenv <someName>
source <someName>/bin/activate
pip install numpy scipy h5py matplotlib

While the virtual environment is active, all calls to python, pip, etc. will be handled by the installation set up for that virtual environment. To leave it, run deactivate. You'll probably want an alias set up to activate your favorite virtual environment.
